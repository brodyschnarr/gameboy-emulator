# Retirement Calculator Changelog

## V4 (In Progress) - Visual Overhaul + Family Mode
### Major Features
- **Interactive Canada Map**: Visual SVG map instead of dropdown, click provinces to drill into regions
- **Regional Benchmarks**: Toronto vs Northern ON, Vancouver vs Rest of BC, etc. - location-specific averages
- **Family Mode**: When "Me + Partner" selected:
  - Income splits into Person 1 + Person 2 (for separate CPP calculations)
  - Life expectancy plans for longer-living spouse
  - Accounts stay combined (household totals)
- **Additional Income Sources**: Dynamic system to add:
  - Defined Benefit Pension
  - Annuity
  - Rental Income
  - Part-Time Work
  - Other Income
  - Each with: Amount, Start Age, End Age
